## Purpose

This document preserves the visual language extracted from the original frontend prototype. The actual frontend implementation is maintained in a separate branch, but this style guide remains the source of visual reference for future UI work.

## Typography

- primary font family: `Raleway`, Arial, sans-serif
- display weight: `800`
- interface weight: `700`
- body weight: `500`
- uppercase labels use expanded letter spacing
- display headings use slightly compressed letter spacing

## Color System

Primary brand colors:

- `brand-ink`: dark neutral for text and surfaces
- `brand-paper`: white base background
- `brand-lime`: primary accent
- `brand-blue`: secondary accent
- `brand-coral`: warning and manual-review accent
- `brand-purple`: reserved tertiary accent

## Layout Principles

- wide dashboard container with responsive padding
- multi-column operational workspace
- summary strips and reviewer panels arranged in clear hierarchy
- information density should remain readable under reviewer workload

## Interaction Patterns

- buttons use short transform and color transitions
- navigation underline uses a scale animation
- queue rows use subtle hover state changes
- visual emphasis should support ranking, review, and override workflows
